# AdventOfCode2018
Solutions to the Advent of Code 2018

## Install
You need [cmake](https://cmake.org/) installed.
## Run
cd to a directory, such as [day01](day01) then build with
```
mkdir build
cd build
cmake -G "Unix Makefiles" ..
make
```
Run executable with
```
./advent-of-code-2018-day-01
```
